How to Run this Project:

Clone the repository.
Get a free Google Maps API Key from Google Cloud Console.
Replace YOUR_GOOGLE_MAPS_API_KEY in android/app/src/main/AndroidManifest.xml and web/index.html with your actual key.

Run flutter pub get and flutter run.
